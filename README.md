# 🤖 ChatSharp — Chat com IA usando ML.NET

Projeto em C# que implementa uma simulação de **chat interativo com inteligência artificial**, utilizando **ML.NET** para classificação de intenções do usuário diretamente no terminal.

---

## 🎯 Objetivo

Este projeto foi desenvolvido para praticar:

- Fundamentos de Machine Learning aplicado
- Uso do ML.NET em aplicações reais
- Processamento de linguagem natural (simples)
- Estruturação de aplicações em C# (.NET)

---

## ⚙️ Funcionalidades

- 💬 Chat interativo via console
- 🧠 Classificação de intenções com ML.NET
- 🤖 Respostas automáticas baseadas na intenção detectada
- 📚 Modelo treinado diretamente no código
- ⚡ Execução simples via terminal

---

## 🔍 Intenções reconhecidas

O modelo consegue identificar:

- Saudação
- Ajuda
- Curiosidade
- Agradecimento
- Frustração
- Comemoração

---

## 🧠 Como funciona

1. O sistema possui um conjunto de frases rotuladas
2. O ML.NET treina um modelo com esses dados
3. Quando o usuário digita uma mensagem:
   - O modelo prevê a intenção
   - O sistema retorna uma resposta correspondente
